# Fertilizer Chatbot

## Project Overview

The Fertilizer Chatbot is a web-based application developed using Java Servlets and deployed on Apache Tomcat 9. The system assists users in obtaining fertilizer recommendations and agricultural guidance through an interactive chatbot interface.

The application processes user inputs on the server side and generates appropriate responses based on predefined logic.

---

## Features

- Interactive chatbot interface
- Fertilizer recommendations
- Crop-related guidance
- Server-side request and response handling
- Web-based deployment

---

## Technologies Used

## Project Structure

- JSP / HTML Pages
- Servlet Classes
- Web Configuration (web.xml)

---

## How to Run the Project

1. Install Apache Tomcat 9.
2. Open the project in NetBeans.
3. Configure Apache Tomcat 9 server in NetBeans.
4. Build and deploy the project.
5. Run the project in the browser using the Tomcat server.
